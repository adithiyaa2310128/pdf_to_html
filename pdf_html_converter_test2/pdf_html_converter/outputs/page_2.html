1. Basic Ping:
ping <host or IP>
This command sends ICMP echo requests to the specified host or IP address and displays the
round-trip time (RTT) for each reply. It continues to send echo requests until interrupted.
2. Specify Number of Echo Requests:
ping -c <count> <host or IP>
The -c option allows you to specify the number of ICMP echo requests to send. It sends the
specified number of requests and stops afterward, displaying the statistics.
