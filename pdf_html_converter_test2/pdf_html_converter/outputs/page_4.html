5. Specify Packet Size:
ping -s <size> <host or IP>
The -s option allows you to specify the size in bytes of the ICMP echo request packet. This can be
used to test the maximum transmission unit (MTU) of the network .
6. To Set Time Limit:
ping -w <size> <hostname>
To stop receiving a ping output after a specific amount of time, add -
w and an interval in seconds to your command.
