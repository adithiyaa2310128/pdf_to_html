DIG:
•
The dig command, short for "Domain Information Groper," is a command-line network
administration tool used to perform DNS (Domain Name System) queries.
•
It is commonly available on Unix-like operating systems, including Linux and macOS.
•
The dig command allows you to retrieve various types of DNS information from DNS
servers.
•
It provides a flexible and powerful way to query DNS records, troubleshoot DNS-related
issues, and gather DNS-related information.
